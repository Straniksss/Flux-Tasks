# Security Policy

## Supported Versions

The latest stable release of Flux Tasks is actively supported with security updates and bug fixes.

Older versions may not receive security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in Flux Tasks, please report it responsibly.

Do not create a public GitHub issue for security vulnerabilities.

Instead, contact the maintainer directly:

Email: [security@tech.bamboo-ecosystem.tech](mailto:security@tech.bamboo-ecosystem.tech)

Please include:

* A description of the vulnerability
* Steps to reproduce
* Potential impact
* Proof of concept if available

## Response Process

Security reports will be reviewed as quickly as possible.

If the issue is confirmed:

1. The vulnerability will be investigated.
2. A fix will be developed and tested.
3. A new release will be published.
4. Credit may be given to the reporter if requested.

Thank you for helping keep Flux Tasks secure.
